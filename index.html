Abstract

  • NAND gate is the UNIVERSAL gate which can be used as any logic gate. As the 4 input NAND gate is
    used for the purpose of buffers, inverters, encoder, decoders, and many more circuit and the CMOS
    used for obtaining the actual output and observing the waveform. A 4 input NAND gate circuit uses
    NAND gate having 4 inputs and 1 output.


Detailed explanation of circuit



  • NAND GATE a Boolean operator which gives the value zero if and only if all the operands have a
    value of one, answer otherwise has a value of one equivalent to NOT AND.


  • PMOS LOGIC FROM P-channel metal oxide semiconductor is a family of digital circuits based on P-
    Channel, enhancement mode


  • NMOS stands for N-type metal oxide semiconductor is a type of MOSFET in which electron are the
    dominant charge carrier in the semiconductor channel.


  • The 4 input CMOS NAND gate we have to use 4 NMOS and 4 PMOS transistors which also having
    single output with VDD and 4 digital input and GND supply. There are 4 digital input that is A,B,C,D
    inputs which are given to the PMOS1 and NMOS1 is given to the input A, and input B is given to the
    PMOS2 and NMOS2, and input C is given to the PMOS3 and NMOS3, and input D is given to the

    • PMOS4 and NMOS4 respectively.


    Truth Table

  • All the 4 PMOS is given to the VDD i.e. PMOS1 is given to the VDD,PMOS2 is also given to the
    VDD,PMOS3 is given to the VDD, PMOS4 also given to the VDD.


  • And all 4CMOS is connected to the ground i.e. NMOS1 is connected to ground, NMOS2 is also
    connected to the ground, NMOS3 is also connected to the ground and NMOS4 is also connected to
    the ground respectively.
